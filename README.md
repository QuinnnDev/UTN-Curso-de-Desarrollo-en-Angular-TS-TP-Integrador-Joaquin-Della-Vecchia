## 🚀 Cómo ejecutar el proyecto

# Metodo 1:
Para correr el proyecto localmente:

1. Abrí una terminal en la carpeta raíz del proyecto  
   (la que contiene `src`, `public`, etc.)

2. Ejecutá el siguiente comando:

```bash
npm run dev
```

# Metodo 1:
